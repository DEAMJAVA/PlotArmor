# Plot Armor

When you're telling a story, the main character isn't supposed to die.

**Plot Armor** is a lightweight server-side mod designed for **scripted Minecraft servers**, roleplay events, adventure maps, and cinematic productions. It allows administrators to grant specific players complete invulnerability, ensuring your story unfolds exactly as planned.

Whether you're running a lore-heavy SMP, filming a machinima, or hosting a live event, Plot Armor keeps your important characters alive until *you* decide otherwise.

## Features

### 🎬 Made for Storytelling

Give plot armor to any player so they can't be killed during important moments.

Ideal for:

* Story-driven SMPs
* Lore events
* Cinematics and machinima
* Adventure maps
* Scripted boss fights
* Live server events

### ⚡ Simple Administration

Manage everything with three straightforward commands:

* `/plotarmor add <player>` — Give a player plot armor.
* `/plotarmor remove <player>` — Remove a player's plot armor.
* `/plotarmor list` — View all players currently protected.
